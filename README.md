✂️ Barbearia API
O Barbearia API é um sistema de gerenciamento de serviços de estética masculina. Ele permite o controle de agendamentos, cadastro de clientes, serviços (corte, barba, etc.) e a gestão de horários dos barbeiros, focando em uma experiência de usuário fluida e sem conflitos de agenda.

🛠️ Tecnologias Chave
Java 17 & Spring Boot 3: Performance e estabilidade.

Spring Data JPA: Mapeamento objeto-relacional eficiente.

H2/PostgreSQL: Flexibilidade para ambiente de teste e produção.

Lombok: Redução de código boilerplate.

Bean Validation: Regras estritas para horários e dados de clientes.

📋 Funcionalidades
Gestão de Serviços: Cadastro de serviços com preços e tempo de duração estimado.

Agendamento Online: Sistema de marcação que evita sobreposição de horários.

Cadastro de Clientes: Histórico de visitas e preferências.
